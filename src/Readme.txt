1. Что означает аннотация @Before?

Аннотация @Before в JUnit (особенно до версии 5) используется для обозначения метода,
который будет выполняться перед каждым тестовым методом. Это удобно для подготовки тестовой среды:
инициализации объектов, установки начальных значений и т. д.

2. Как в тесте проверить, что метод должен бросить исключение?

В JUnit 5 это можно сделать с помощью метода assertThrows.
Этот метод проверяет, что в определенном коде выбрасывается исключение указанного типа.

3. Что такое mock? Spy?
Мock — объект, который используется для имитации поведения зависимостей тестируемого класса.
С помощью mock можно задавать заранее определённые ответы методов, чтобы протестировать только логику тестируемого класса.

Spy — объект, который позволяет отслеживать вызовы реального объекта.
Spy использует настоящий объект, но вы можете переопределить некоторые его методы.

4. Для чего применяется статический метод Mockito.verify?

Метод Mockito.verify используется для проверки того, что методы объекта были вызваны с определёнными параметрами.
